# 🧩 Ejercicio 1: Secuencia de Números Especiales

Enunciado:

Se dice que un número es especial si cumple estas condiciones:

No es divisible por 2 ni por 3.

La suma de sus dígitos también es impar.

Escribe un programa que:

Reciba un número entero N (mayor que 0).

Genere los primeros N números especiales.

Imprima la lista de esos números y la suma total de todos ellos.

ejemplo: 
```js
Entrada: N = 5
Salida: 
Números especiales: [1, 5, 7, 11, 13]
Suma total: 37
```
