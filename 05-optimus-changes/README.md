# 🧩 Ejercicio 5: Cambio Óptimo

## 📜 Enunciado

Dado un conjunto de monedas y un valor objetivo `X`, debes encontrar:

1. La **mínima cantidad de monedas** necesarias para alcanzar ese valor.
2. Mostrar qué monedas se usaron.

---

## 🧪 Ejemplo

Monedas: [1, 3, 4]  
Valor: 6  

Salida:
Cantidad mínima: 2  
Monedas usadas: [3, 3]

---

## ⚠️ Restricciones

### 🐍 Python
- ❌ No usar arrays auxiliares grandes
- ✅ Usa recursión + memoización

### ☕ Java
- ❌ No usar recursión
- ✅ DP iterativo

### ⚡ JavaScript
- ❌ No usar `for`
- ✅ Usa `reduce`, recursión o funciones

### 🟦 TypeScript
- ✅ Tipar completamente la solución
- ❌ No usar estructuras sin tipo

---

## 🧠 Habilidad

- Programación dinámica
- Optimización
